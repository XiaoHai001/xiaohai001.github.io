  2. Git工具：用于执行命令。

  3. hugo工具

开始制作个人博客：

生成博客文件夹 cmd中操作语句: hugo new site myblog (生成文件夹为myblog,为你的hugo站点)

浏览器进入：https://themes.gohugo.io/查看你想要的主题；然后进入

cmd进入myblog目录：

输入你复制好的代码；

启动博客：

hugo server -t m10c –buildDrafts

新建blog.md（写日志的文本名称）博客文章：hugo new post/blog.md （最好是这样写，也可直接用markdown创建，但需要改头部）==但必须在post文件夹下==

title中编写标题

将个人博客部署到远端服务器（GitHub）

新建仓库；

==Repository名字必须是自己昵称全小写（前面的）.github.io;==（这里我之前已经创建了的）

然后点确定；

cmd 命令进入myblog输入命令：hugo –theme=m10c –baseUrl="https:xiaohai001.github.io/” –buildDrafts

进入public文件夹：cd public/

命令：git init

添加所有：git add .

提交命令：git commit -m “提交上去的名字“

public文件夹与本地关联命令：git remote add origin https://github.com/xiaohai002.github.io.git

推到master上：git push -u origin master

输入自己的用户号和密码

图片可以放到github的仓库中，然后取外链地址，这样比放在项目中提交的情况好，但是这时候github保存图片存在dns污染，

解决链接：https://blog.csdn.net/qq_38232598/article/details/91346392
